# 浏览器调用接口示例
```
主要用于各种查询----区块，交易，地址等
```

测试公网
```
url：http://test-system.htdfscan.com
```
主网：
```
url：http://www-system.htdfscan.com 
```

### 查询某一高度的区块

**方法**：   GET
```
~$ curl http://test-system.htdfscan.com/api/block?blockheight=1000
```
**返回** :


```
{
	"code": 200,
	"block": [{
		"id": 874152,
		"blockheight": 1000,
		"blockhash": "AC1D1A5607C56C0219CF27B0A53DEF5C8F347AA1D534A7D3A4AB5A801F70B251",
		"outtime": "2019-07-23 15:53:09",
		"count": 0,
		"num": 0
	}],
	"pagenum": 98104
}
```
